Mastermind

The idea of the game is for one player (the code-breaker) to guess the secret code chosen by the computer (the code-maker).

Rules

1. At the start of the game the computer will randomly select a pattern of different numbers. You need to guess the combination of numbers

2. Type your guess numbers into the input form and press check

"Attempt 10"

3.You will have limited number of attempts to guess the number combinations (depending on your game settings).

4. You'll able to see the history of guesses and their feedback.

5. Each guess number has its own color.

6. Computer will provide one of the following response as feedback:

  • The player had guessed a correct number and its correct location - green square
  • The player had guess a correct number - orange square
